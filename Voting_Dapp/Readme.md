

# 📢 Voting DApp

A decentralized voting platform built on Ethereum, allowing users to participate in a transparent and secure voting process. The project leverages **Solidity smart contracts**, **MetaMask** for wallet integration, and is deployed using **Vercel** for hosting.

---

## 🚀 Features

* User login via **MetaMask** wallet.
* Display connection status and connected MetaMask account address.
* Real-time voting interface showing:

  * Remaining time for voting.
  * Voting status ("Already Voted").
  * List of candidates with their vote counts.
* Secure and trustless voting powered by smart contracts.

## 🏁 Getting Started

Follow these steps to set up and run the project locally.

### 1️⃣ Install the Project

Clone the repository:
git clone [https://github.com/RogerZoe/Dapps.git](https://github.com/RogerZoe/Dapps.git)
cd Dapps/Voting-DApp


---
 ### Install dependencies:

```bash
npm install
```


### 2️⃣ Running the Project

Start the development server:

```bash
npm run dev
```
